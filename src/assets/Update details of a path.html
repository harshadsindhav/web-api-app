UPDATE DETAILS OF A PATH

Overview

Updates the details of a path based on the Path's ID and Certification's ID passed as parameter values.

Requires OAuth

No

Method

PUT

URL

https://<hostname-api.sabacloud.com>/v1/path/:id

Calling Options
Table 1. Calling Options Calling Options
Name Description Sample Value Data Type Required?
id Path's ID string Yes
certification_id Certification's ID string Yes
isrecertification_track Is it a recertification track false string No
name Name of the path Path101 modified string Yes
is_default_path Is is a default path true string Yes
customValues Custom Values string No
Request Body
{
   "name": "Path modifieddd",
   "isrecertification_track": false,
   "certification_id": "crtfy000000000200784",
   "is_default_path": true,
   "id": "track000000000200021"
}
