CREATE A NEW STATE

Overview

Allows creating a new State.

Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/state

Calling Options
Table 5. Calling Options Calling Options
Name Description Default Value Data Type Required?
name The name of the State State1 string Yes
isoCode The ISO code for the state IS string Yes
shortName The short name for the state string No
country The country for the state {\"id\": \"cntry000000000001000\", \"displayName\":\"new1\"} string Yes
Request Body
{
 "name": "Maha",
 "shortName":"mah",
 "isoCode":"mh",
 "country":{"id":"cntry000000000001000"}
}
Return Values
{
    "id": "state000000000003490",
    "displayName": null,
    "href": "http://<host_name>/v1/state/state000000000003490"
}
