ASSIGN POSITION FOR A PERSON

Overview

Adds positions for a particular person based on the person's ID that is passed as a parameter value.

Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/people/:personId/positions
