GET ALL ORDERS

Overview

Returns all Orders for the last three months only. You can use FIND DETAILS OF ORDERS API to get orders for the months beyond the three months range by providing the created_on search criteria.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/learning/order?count=:count&startPage=:startPage

Calling Options
Table 1. Calling Options Calling Options
Name Description Sample Value Data Type Required?
count The number of records per page. 10 string No
startPage The start page number for the list of records. 1 string No
